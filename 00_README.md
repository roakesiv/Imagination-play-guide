# Magic Creature Card Maker

A simple static prototype for parent-guided creative play with kids ages 3-6.

Parents ask structured questions, type the child's exact answers, and generate a copy/paste-ready ChatGPT image prompt for a kid-friendly magical creature profile card. The prototype is testing whether the guided prompts make it fast and low-pressure to capture wonderfully specific kid ideas without interpreting or improving them.

## How to Run

Open `index.html` in any modern browser. No build step, backend, login, storage, packages, or image API are required.

## Prototype Flow

1. Read the short intro.
2. Ask the child each question and enter their raw answers.
3. Tap `Make Prompt`.
4. Copy the generated image prompt with `Copy Prompt`.
5. Review the creature summary and detail recap.
6. Tap `Reset` to make another creature.

Use `Fill Example` to quickly populate a sample creature for testing.
