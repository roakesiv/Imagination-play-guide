# P4.2 Prompt Discovery - Activity Page Outputs

**Project:** Magic Creature Card Maker / Imagination Play Guide  
**Prototype:** P4.2  
**Status:** Complete / ready for implementation review  
**Purpose:** Capture manual prompt mockups before Codex implementation.

---

## 1. Discovery Goal

Discover prompt structures for P4.2 printable activity pages.

The goal was to validate the prompt patterns manually before adding them to the app.

---

## 2. Discovery Process

Discovery used this loop for each output:

```text
draft prompt
-> generate mockup in ChatGPT or target image tool
-> assess result
-> revise prompt
-> repeat until good enough
-> mark validated template
-> hand implementation to Codex
```

## 4. Maze Discovery

### 4.1 Success Criteria

A good Maze page should have:

- black-and-white printable layout
- creature clearly at the start
- home/object/friend clearly at the finish
- easy-to-follow maze path
- thick readable maze walls/lines
- simple enough for young children
- no tiny corridors
- not too hard
- minimal visual clutter
- optional simple decorations that can be colored
- kid-safe whimsical feel

### 4.2 Prompt Version A

```text
Create a black-and-white printable maze activity page for a young child featuring a kid-friendly magical creature.

Main creature: [Name], a [Creature Mix]
Magic: [Magic]
Home/background: [Home]
Personality: [Personality]
Extra detail: [Extra Detail]

Make [Name] the clear starting character at the beginning of the maze. Put [Name]'s home, a simple [Home], as the clear goal at the end of the maze.

Create a simple maze path that helps [Name] travel from the start to the home. The maze should be easy enough for a young child, with thick readable lines, wide paths, and no tiny corridors.

Include a few simple, easy-to-color decorations from [Name]'s world, such as items related to the home, magic, accessories, or extra detail. Keep the page uncluttered and easy to use.

Use:
- clean black-and-white line art
- bold, readable outlines
- wide maze paths
- simple shapes
- open spaces for coloring
- minimal detail
- no shading or grayscale
- no tiny patterns
- no confusing overlapping paths
- no scary details

Make it feel like a cute children's maze worksheet and coloring page, not a difficult puzzle book page or an adult maze.
```

|---|---|---|---|---|

### 4.4 Validated Maze Template

Prompt Version A is the validated Maze template for P4.2 WP1.

### 4.5 Maze Implementation Notes

- Add only after prompt is validated.
- Keep bridge UI minimal.
- Use existing creature data.
- Use shared missing-field fallback.
- If creature name is missing, use "the magic creature" in surrounding prompt text.
- Do not add activity-pack behavior.

## 5. Letter Tracing Discovery

### 5.1 Success Criteria

A good Letter Tracing page should have:

- black-and-white printable worksheet layout
- large traceable letters
- dotted or dashed tracing guides
- creature name or fallback word
- simple creature illustration
- clear writing practice area
- not too many words
- suitable for preschool and early-reader/kindergarten cohorts
- enough open space
- no tiny decorations
- kid-safe playful feel

### 5.2 Prompt Version A

```text
Create a black-and-white printable letter tracing worksheet for a young child featuring a kid-friendly magical creature.

Main creature: [Name], a [Creature Mix]
Magic: [Magic]
Home/background: [Home]
Personality: [Personality]
Extra detail: [Extra Detail]

Make the worksheet focus on tracing the creature's name: [UPPERCASE_NAME].

At the top of the page, show [Name]'s name clearly in large friendly letters.

Include large dotted or dashed traceable letters for:
[UPPERCASE_NAME]

Add a simple, cute black-and-white line art picture of [Name] near the name. Keep the creature easy to color, with bold clean outlines, simple shapes, and minimal detail.

Include a few simple decorations from [Name]'s world, such as items related to the home, magic, accessories, or extra detail, but keep the page uncluttered and easy to use.

Use:
- clean black-and-white line art
- large traceable letters
- dotted or dashed letter guides
- bold readable outlines
- simple shapes
- open spaces for coloring
- minimal detail
- no shading or grayscale
- no tiny patterns
- no cluttered background
- no scary details

Make it feel like a cute preschool handwriting worksheet and coloring page, not a complex school worksheet or adult coloring page.
```

- clear pattern rows

- minimal detail
- no shading or grayscale
