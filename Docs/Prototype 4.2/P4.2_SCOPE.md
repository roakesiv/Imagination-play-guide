**Project:** Magic Creature Card Maker / Imagination Play Guide  
**Prototype:** P4.2  
**Status:** WP1-WP5 implemented / smoke tested  
**Prototype type:** Activity output expansion

---

## 1. Goal

Add validated printable activity-page output types as complete vertical slices:

1. Maze
2. Letter Tracing
3. Count the Objects
4. Find the Letter
5. Draw the Missing Detail

Each output should have a prompt structure good enough for first real testing before implementation.

Prompt mockup discovery is complete for WP1-WP5. Maze, Letter Tracing, Count the Objects, Find the Letter, and Draw the Missing Detail mockups passed.

---

## 2. Learning Question

> Can more activity pages be generated from the existing creature attributes using focused, kid-appropriate prompt templates?

Secondary question:

> Can the V4.0 architecture support adding new prompt outputs without tangling UI behavior, prompt engineering text, and editable content?

---

## 3. Product Reason

V4.1 proved that prompt quality improvements make immediate impact for printable outputs. P4.2 proved that adding activity-page outputs is easy enough after the V4.0 architecture runway.

P4.2 continues while the rapid activity-page pattern stays lightweight and useful for near-term kid testing.

---

## 4. In Scope

- Validated prompt templates.

- New output templates in `promptTemplates.js`.
- Existing creature data contract used.
- Existing prompt builder helpers used.
- Minimal UI changes only to expose new output options.
- Docs and manual smoke test.

Current P4.2 outputs:

- Maze
- Letter Tracing
- Count the Objects
- Find the Letter
- Draw the Missing Detail

---

## 5. Out of Scope

Do **not** add in P4.2:

- Bridge grouping.
- Step navigation.
- Page split.
- Activity pack generation.
- PDF export.
- Printing workflow.
- Direct AI image generation.
- Saved outputs.
- New activity types beyond focused, validated P4.2 activity-page work packages.
- New form fields unless proven necessary during mockup discovery.
- Story mode.
- Parent DM assistant.
- Backend, accounts, auth, storage, database, packages, build step, or framework.
- Major visual redesign.

## 8. Prompt Requirements

### Count the Objects

- black-and-white printable counting activity page
- young-child friendly
- uses current creature data
- asks the child to count objects
- includes 3 to 5 counting groups
- uses large clear objects
- includes simple blank answer spaces
- avoids clutter, shading, grayscale, tiny patterns, and scary details

### Find the Letter

- black-and-white printable letter-finding worksheet
- young-child friendly
- uses the first letter of the creature name
- falls back to letter `M` and word `MAGIC` if name is missing
- includes big target letter near the top
- asks the child to find and circle the target letter
- uses large readable letters
- avoids tiny letters and clutter

### Draw the Missing Detail

- black-and-white printable drawing activity page
- young-child friendly
- includes large open blank drawing area
- includes the prompt `Draw something magical for [Name]!`
- keeps most of the page open
- avoids clutter, shading, grayscale, tiny patterns, and scary details
