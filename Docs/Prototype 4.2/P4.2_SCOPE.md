## 2. Learning Question

> Can more activity pages be generated from the existing creature attributes using focused, kid-appropriate prompt templates?

Secondary question:

> Can the V4.0 architecture support adding new prompt outputs without tangling UI behavior, prompt engineering text, and editable content?

---

## 3. Product Reason

V4.1 proved that prompt quality improvements make immediate impact for printable outputs. P4.2 proved that adding activity-page outputs is easy enough after the V4.0 architecture runway.

P4.2 continues while the rapid activity-page pattern stays lightweight and useful for near-term kid testing.

---

- Maze

---

## 5. Out of Scope

Do **not** add in P4.2:

- Bridge grouping.
- Step navigation.
- Page split.
- Activity pack generation.
- PDF export.
- Printing workflow.
- Direct AI image generation.
- Saved outputs.
- New activity types beyond focused, validated P4.2 activity-page work packages.
- New form fields unless proven necessary during mockup discovery.
- Story mode.
- Parent DM assistant.
- Backend, accounts, auth, storage, database, packages, build step, or framework.
- Major visual redesign.

---

## 6. Known Future UX Need

Bridge grouping is expected later.

Current bridge outputs may become too many. Do not solve this in P4.2 unless the bridge becomes unusable.

Story and Adventure remain in the codebase but are disabled in the active bridge UI until V5 story prompt work.

## 8. Prompt Requirements

### Trace the Path

- black-and-white printable tracing path worksheet
- young-child friendly
- includes 3 to 5 large traceable paths
- includes dotted or dashed tracing paths
- includes wide spacing
- uses simple prewriting shapes
- avoids shading, grayscale, clutter, tiny patterns, and scary details

### Matching Page

- black-and-white printable matching activity page
- young-child friendly
- includes two clear columns
- includes 4 to 6 matching object pairs
- asks child to draw lines to match
- uses large simple recognizable objects
- avoids shading, grayscale, clutter, tiny patterns, and scary details

### Finish the Pattern

- black-and-white printable pattern activity page
- young-child friendly
- includes 3 to 4 pattern rows
- uses large easy-to-recognize objects
- includes simple preschool-friendly patterns
- includes blank spaces for answers
- includes `Finish the magical patterns!`
- avoids shading, grayscale, clutter, tiny patterns, and scary details

## 10. Done-Enough Definition

They do not need to be perfect.

Prompt quality can continue improving in later activity-book refinement if testing reveals issues.
