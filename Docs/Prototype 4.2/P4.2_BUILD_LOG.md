## What Changed

### WP1 - Maze

- Added a Maze bridge output card.
- Added `artifacts.maze` in `promptTemplates.js`.
- Uses current creature data and `the magic creature` name fallback.

### WP2 - Letter Tracing

- Added a Letter Tracing bridge output card.
- Added `artifacts.letterTracing` in `promptTemplates.js`.
- Uses uppercase creature name as tracing word.
- Uses `MAGIC` tracing fallback when creature name is missing.

### WP3 - Count the Objects

- Added a Count the Objects bridge output card.
- Added `artifacts.countObjects` in `promptTemplates.js`.
- Requests a black-and-white printable counting activity page for a young child.
- Requests 3 to 5 counting groups, large clear objects, and blank answer spaces.

### WP4 - Find the Letter

- Added a Find the Letter bridge output card.
- Added `artifacts.findLetter` in `promptTemplates.js`.
- Uses the first letter of the creature name.
- Uses fallback letter `M` and fallback word `MAGIC` when creature name is missing.
- Requests a big target letter near the top and large readable letters to find and circle.

### WP5 - Draw the Missing Detail

- Added a Draw the Missing Detail bridge output card.
- Added `artifacts.drawMissingDetail` in `promptTemplates.js`.
- Requests a black-and-white drawing activity page with a large open blank drawing area.
- Includes the prompt: `Draw something magical for [Name]!`

### WP6 - Trace the Path

- Added a Trace the Path bridge output card.
- Added `artifacts.tracePath` in `promptTemplates.js`.
- Requests a black-and-white printable tracing path worksheet for a young child.
- Requests 3 to 5 large dotted or dashed traceable paths with simple prewriting shapes and wide spacing.

### WP7 - Matching Page

- Added a Matching Page bridge output card.
- Added `artifacts.matchingPage` in `promptTemplates.js`.
- Requests a black-and-white printable matching activity page for a young child.
- Requests two clear columns, 4 to 6 matching object pairs, and space for drawing matching lines.

### WP8 - Finish the Pattern

- Added a Finish the Pattern bridge output card.
- Added `artifacts.finishPattern` in `promptTemplates.js`.
- Requests a black-and-white printable pattern activity page for a young child.
- Requests 3 to 4 preschool-friendly pattern rows, blank answer spaces, and the prompt `Finish the magical patterns!`

### Bridge State - Pause Story/Adventure Bridge Buttons

- Disabled the Story and Adventure bridge buttons in `index.html`.
- Updated their helper text to show they are paused for V5 story work.
- Added disabled-card styling in `styles.css`.
- Preserved the Story and Adventure templates in `promptTemplates.js` for future reactivation.
