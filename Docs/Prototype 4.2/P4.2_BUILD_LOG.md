## What Changed

### WP1 - Maze

- Added a Maze bridge output card.
- Added `artifacts.maze` in `promptTemplates.js`.
- Uses current creature data and `the magic creature` name fallback.

### WP2 - Letter Tracing

- Added a Letter Tracing bridge output card.
- Added `artifacts.letterTracing` in `promptTemplates.js`.
- Uses uppercase creature name as tracing word.
- Uses `MAGIC` tracing fallback when creature name is missing.

### WP3 - Count the Objects

- Added a Count the Objects bridge output card.
- Added `artifacts.countObjects` in `promptTemplates.js`.
- Requests a black-and-white printable counting activity page for a young child.
- Requests 3 to 5 counting groups, large clear objects, and blank answer spaces.

### WP4 - Find the Letter

- Added a Find the Letter bridge output card.
- Added `artifacts.findLetter` in `promptTemplates.js`.
- Uses the first letter of the creature name.
- Uses fallback letter `M` and fallback word `MAGIC` when creature name is missing.
- Requests a big target letter near the top and large readable letters to find and circle.

### WP5 - Draw the Missing Detail

- Added a Draw the Missing Detail bridge output card.
- Added `artifacts.drawMissingDetail` in `promptTemplates.js`.
- Requests a black-and-white drawing activity page with a large open blank drawing area.
- Includes the prompt: `Draw something magical for [Name]!`
