## 8. Testing Scope Lesson

The WP2 automated Edge smoke test eventually worked, but the effort was disproportionate for the current app architecture. The durable lesson is that testing should scale with the current NFRs. For this static, no-build prototype, manual browser smoke testing plus later mobile/play-trial validation is the right default. Automated browser testing should wait until a future technical runway explicitly supports it.
