# P4.1 Prompt Discovery

**Project:** Magic Creature Card Maker / Imagination Play Guide
**Prototype:** P4.1
**Focus:** Printable activity prompt quality

---

## Discovery Summary

Manual prompt mockups in ChatGPT showed that the new Coloring Page structure worked better than the prior prompt for guiding printable activity output.

The improved structure is more explicit about:

- black-and-white printable output
- a young-child audience
- the creature as the clear main focus
- simple background elements
- bold clean outlines
- large simple shapes
- open coloring spaces
- avoiding visual complexity
- avoiding adult coloring page style

## WP1 Validated Coloring Page Structure

```text
Create a black-and-white printable coloring page for a young child featuring a kid-friendly magical creature.

Main creature: [Name], a [Creature Mix]
Magic: [Magic]
Home/background: [Home]
Personality: [Personality]
Extra detail: [Extra Detail]

Show [Name/the creature] as the clear main focus of the page. Include a simple, playful background with a few easy-to-color elements inspired by the home/background, but keep the scene uncluttered and easy for a child to color.

Use:
- bold, clean outlines
- large simple shapes
- open spaces for coloring
- minimal detail
- no shading or grayscale
- no tiny patterns
- no heavy textures
- no scary details

Make it feel like a cute children's coloring book page, not a detailed illustration or an adult coloring page.
```

## Implementation Decision

Implement this as WP1 by updating only `promptTemplates.js` `artifacts.coloring`.

Do not update Find-It until a later P4.1 work package is reviewed and approved.
