# Magic Creature Card Maker — Design Documentation

## Project purpose

This project is a learning prototype and family-use tool. It is being used to practice Stanford-style design thinking, rapid prototyping, and AI-assisted software development while building something genuinely useful for parent-led creative play with young children.

## Current product framing

A phone-friendly parent-guided creative play app that helps young kids invent absurd magical creatures, captures their answers as raw text, and generates a well-structured, kid-safe prompt for ChatGPT image generation.

The immediate artifact is a magical creature profile card image. The longer-term product vision is a guided co-storytelling tool that helps a parent “DM” imaginative adventures with kids and eventually turn those sessions into simple illustrated stories.

## Current prototype phase

The project is ready to move from discovery into Prototype 3.

Prototype 3 should build a simple static web app that:

1. Guides a parent through creature-creation questions.
2. Captures the child’s raw, weird, specific answers.
3. Generates a copy/paste-ready prompt for ChatGPT.
4. Generates a short creature summary for future story use.
5. Displays a clean attribute recap.

No backend, no image API, no login, no save system, and no story mode yet.

## Core design rule

Preserve the weird.

The child’s absurd specificity is the delight. The app should provide structure, not overwrite the child’s imagination.
