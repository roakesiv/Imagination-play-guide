# P4.4 Batch Prompt Mockup

**Project:** Magic Creature Card Maker / Imagination Play Guide  
**Prototype:** P4.4  
**Work Package:** WP2 — Batch Prompt Pack Discovery  
**Status:** Complete - discovery succeeded

---

## 1. Mockup Goal

Test whether one batch prompt can reduce repeated copy/paste work.

Instead of copying 10 individual activity prompts, the parent copies one batch prompt per creature.

ChatGPT should:

1. generate Page 01 only,
2. stop,
3. wait for `NEXT`,
4. generate the next page,
5. preserve the page order.

---

## 2. Test Questions

After trying the batch prompt, answer:

1. Does ChatGPT generate only Page 01 first?
2. Does saying `NEXT` work?
3. Does ChatGPT preserve page order?
4. Does image quality stay acceptable?
5. Does the batch prompt reduce stress?
6. Does suggested filename text help organization?
7. Does same-thread context make later pages worse?
8. Does this beat copying 10 prompts individually?

---

## 3. Batch Prompt Example — Amor

```text
I am creating a printable activity pack for young children.

Use this creature:

Name: Amor
Creature: raccoon witch + pony
Magic: invisibility + water
Colors: purple and blue
Home/background: castle of candy and balloons
Personality: kind and brave
Accessories: books and toys
Extra detail: loves storytime

Task:
Create the activity pages one at a time.

Important:
- Generate only one image at a time.
- Start with Page 01.
- After each image, stop and wait for me to say NEXT.
- When I say NEXT, generate the next page.
- Use black-and-white printable worksheet style.
- Use bold clean outlines.
- Keep pages simple for young children.
- Avoid grayscale, shading, clutter, tiny details, scary elements, and adult worksheet complexity.
- Before each image, show the page title and suggested filename.

Activity Pack Order:

Page 01 — Coloring Page
Suggested filename: amor_01_coloring_page.png
Create a black-and-white printable coloring page for a young child featuring Amor as the clear main focus. Include a simple castle of candy and balloons background. Use bold clean outlines, large simple shapes, open spaces for coloring, minimal detail, no shading or grayscale, no tiny patterns, no heavy textures, and no scary details. Make it feel like a cute children's coloring book page, not an adult coloring page.

Page 02 — Find-It Game
Suggested filename: amor_02_find_it_game.png
Create a black-and-white printable hidden-object activity page for a young child. Show Amor clearly and make Amor's name easy to read at the top. Create a simple scene in the castle of candy and balloons. Include a bottom "Things to Find" row with picture icons and item names for pre-reading children. Hide books and toys, a storytime detail, an invisibility + water sparkle, a tiny gem, and a small castle sign. Keep it readable, uncluttered, easy to print, and easy to color.

Page 03 — Maze
Suggested filename: amor_03_maze.png
Create a black-and-white printable maze activity page for a young child. Put Amor at the start and the castle of candy and balloons at the finish. Use a simple maze path with thick readable lines, wide paths, no tiny corridors, no confusing overlapping paths, and simple decorations that can be colored.

Page 04 — Letter Tracing
Suggested filename: amor_04_letter_tracing.png
Create a black-and-white printable letter tracing worksheet for a young child. Focus on tracing the creature's name: AMOR. Show Amor's name clearly at the top in large friendly letters. Include large dotted or dashed traceable letters for AMOR. Add a simple cute black-and-white line art picture of Amor with bold outlines and minimal detail.

Page 05 — Count the Objects
Suggested filename: amor_05_count_objects.png
Create a black-and-white printable counting activity page for a young child. Show Amor in a simple playful scene. Add 3 to 5 groups of large easy-to-count objects from Amor's world, such as books, toys, balloons, tiny gems, water sparkles, and candy pieces. Include a simple blank answer space for each group.

Page 06 — Find the Letter
Suggested filename: amor_06_find_letter.png
Create a black-and-white printable letter-finding worksheet for a young child. Focus on the letter A. Show Amor clearly with a big letter A near the top. Create a simple activity where the child finds and circles all the letter A hidden among other large friendly letters. Use large readable letters and no tiny clutter.

Page 07 — Draw the Missing Detail
Suggested filename: amor_07_draw_missing_detail.png
Create a black-and-white printable drawing activity page for a young child. Show Amor clearly in simple line art. Leave one large open blank area where the child can draw an extra magical detail, accessory, snack, toy, sparkle, book, water magic, balloon, candy, or friend for Amor. Add the prompt: "Draw something magical for Amor!"

Page 08 — Trace the Path
Suggested filename: amor_08_trace_path.png
Create a black-and-white printable tracing path worksheet for a young child. Help Amor follow large dotted paths to reach the castle of candy and balloons. Include 3 to 5 large traceable paths: straight, wavy, zigzag, loop-de-loop, and curved rainbow path. Use wide spacing and no clutter.

Page 09 — Matching Page
Suggested filename: amor_09_matching_page.png
Create a black-and-white printable matching activity page for a young child. Show Amor clearly. Use two clear columns of large simple objects from Amor's world. Include 4 to 6 matching pairs such as book, toy, balloon, tiny gem, water sparkle, candy piece, and storytime book. Leave enough space for the child to draw matching lines.

Page 10 — Finish the Pattern
Suggested filename: amor_10_finish_pattern.png
Create a black-and-white printable pattern activity page for a young child. Show Amor clearly. Include 3 to 4 simple pattern rows using large objects from Amor's world, such as balloon, book, gem, water sparkle, candy, toy, and storytime book. Leave a blank space at the end of each row. Include the prompt: "Finish the magical patterns!"

Start now with Page 01 only.
```
